DETECTION METHODS (8 types)
============================

1. Hidden Text Detection
   - Font size 0px or < 2px
   - Opacity 0 or near-zero  
   - CSS visibility/display hidden
   - Text matching background color
   - Off-screen positioning

2. Tracking Pixels
   - 1x1 images
   - Tiny elements (< 10px)

3. Suspicious Links
   - JavaScript URLs
   - Data URLs
   - URL vs text mismatch

4. Homograph Attacks
   - Cyrillic lookalikes (Ð° vs a)

5. Invisible iFrames
   - Hidden iframes
